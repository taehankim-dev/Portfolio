# 영농일지 (Farming Diary)

## 1. 프로젝트 요약
- 자회사 '사라팜'의 스마트팜을 이용하는 고객들이 일자별 작업 내역과 현장 상황을 기록할 수 있는 웹 기반 영농 기록 서비스
- 스마트팜 운영 과정의 작업 이력과 현장 정보를 체계적으로 기록하여 농장 운영 흐름을 한눈에 파악하고, 장기적으로 AI 분석 및 자동화 확장을 위한 데이터 수집 기반 구축
- 프론트엔드 및 백엔드 전체 개발 담당 (풀스택 개발)

## 2. 내가 맡은 역할
- 프론트엔드 및 백엔드 전반 구현
- 농장 운영 도메인 분석을 기반으로 한 기능 설계

## 3. 핵심 경험
- **모바일 최적화된 하이브리드 앱 개발**: 현장에서 사용하기 편리하도록 HEIC 이미지 자동 변환, 클라이언트 사이드 이미지 압축, 터치 친화적 UI를 구현하여 농장 현장에서 실시간으로 작업 내역을 기록할 수 있는 환경 제공
- **AI 확장을 고려한 데이터 구조 설계**: 향후 AI 분석 및 자동화 확장을 고려한 데이터 구조로 설계


## 4. 사용 기술
- **Frontend**: Vue 3, TypeScript, Vite, Pinia, Tailwind CSS, Capacitor (iOS/Android)
- **Backend**: Spring Boot 3, Java 17, MyBatis, JPA/Hibernate, PostgreSQL, JWT, WebSocket
- **Infrastructure**: Docker

## 5. 비고
- 내부 테스트 단계에서 사용성 개선 논의 이후 운영 확장 중단
- AI 확장을 고려한 데이터 구조 설계 경험 축적
