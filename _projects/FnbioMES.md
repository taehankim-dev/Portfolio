# FNBio MES

## 1. 프로젝트 요약
- 에프엔바이오 제조 현장을 위한 제조실행시스템
- 원부자재 입출고, 제품 생산, 칭량 관리, 주문 처리 등 제조 현장 운영에 필요한 기능을 제공하는 내부 시스템
- 프론트엔드 및 백엔드 전체 개발 담당 (풀스택 개발)

## 2. 내가 맡은 역할
- 프론트엔드 및 백엔드 전반 구현
- 제조 도메인 분석을 기반으로 한 기능 설계

## 3. 핵심 경험

- **제조 도메인 기반 MES 핵심 기능 구현**
  원부자재 입출고, 제품 생산, 칭량 기록서, 수율 계산 등
  제조 현장의 주요 업무 흐름을 시스템으로 구조화하고 구현.
  BOM 정보를 기반으로 원료 소진과 생산 데이터를 자동 연계하여
  현장 데이터의 일관성과 무결성을 확보함.

- **프론트엔드–백엔드 데이터 흐름 설계 및 구현**
  Vue 3 기반 프론트엔드와 Spring Boot 백엔드 간의 API 구조를 설계하고,
  복잡한 제조 데이터(칭량 기록서, 생산 이력, 입출고 내역)를
  안정적으로 처리할 수 있도록 화면–API–DB 흐름을 구성.

- **협업 기반 역할 분리 경험**
  LOT 기반 추적 로직과 태블릿 UI는 동료가 담당하고,
  그 외 MES 전반 기능과 PC 기반 관리자 화면, 핵심 비즈니스 로직을 담당.
  역할을 명확히 분리한 협업을 통해 전체 시스템 완성도를 높임.


## 4. 사용 기술
- **Frontend**: Vue 3, TypeScript, Vite, Pinia, AG-Grid, Highcharts, WebSocket (STOMP), Axios, XLSX
- **Backend**: Spring Boot 3.2.4, JPA, MyBatis, PostgreSQL, JWT, WebSocket (STOMP), Apache POI

## 5. 비고
- 제조 현장의 실시간 모니터링과 품질 추적성 확보에 중점을 둔 내부용 시스템
