> **프로젝트 개요**
>
> 웹 기반 MES(Manufacturing Execution System) 프론트엔드 개발 프로젝트입니다. Vuetify를 활용한 UI 퍼블리싱과 데이터 시각화 기능을 구현했으며, Atomic Design 개념을 참고하여 재사용 가능한 컴포넌트 구조를 설계했습니다.
>
> **프로젝트 성격**: 체계적인 컴포넌트 설계를 통해 유지보수성과 재사용성을 확보한 UI 개발 프로젝트

- Atomic Design 참고하여 컴포넌트 구조 개선

- Vuetify 기반 일관된 디자인 시스템

## 기술적 판단 & 설계 포인트

### Atomic Design 참고

컴포넌트 재사용성과 유지보수성을 높이기 위해 Atomic Design 개념을 참고하여 컴포넌트를 계층적으로 구조화했습니다.

**컴포넌트 구조**:
```
src/components/
├── atoms/           # 가장 작은 단위
│   ├── Button.vue
│   ├── Input.vue
│   └── Badge.vue
├── molecules/       # Atom의 조합
│   ├── SearchBar.vue
│   ├── FormField.vue
│   └── StatusCard.vue
├── organisms/       # Molecule의 조합
│   ├── DataTable.vue
│   ├── ProductionChart.vue
│   └── Header.vue
└── templates/       # 페이지 레이아웃
    ├── DashboardLayout.vue
    └── FormLayout.vue
```

**재사용 가능한 SearchBar 컴포넌트 예시**:

Atom 컴포넌트들을 조합하여 검색바를 구성하고, 여러 페이지에서 재사용 가능하도록 설계했습니다.

```vue
<template>
  <div class="search-bar">
    <v-text-field
      v-model="searchQuery"
      placeholder="검색..."
      prepend-inner-icon="mdi-magnify"
      variant="outlined"
      density="compact"
      @keyup.enter="handleSearch"
    />
    <v-btn 
      color="primary"
      @click="handleSearch"
    >
      검색
    </v-btn>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue';

const searchQuery = ref('');

const emit = defineEmits<{
  search: [query: string];
}>();

const handleSearch = () => {
  emit('search', searchQuery.value);
};
</script>
```
