> **프로젝트 개요**
>
> 협업사에서 제작한 스마트팜 미니어처를 실제로 동작시키기 위한 라즈베리파이/아두이노 기반 제어 시스템입니다. 자회사 박람회 전시 일정에 맞춰 미니어처 내 물 분사, 도어 개폐, LED, 환기 팬 등을 제어하는 코드를 구현했습니다.
>
> **프로젝트 성격**: 박람회 일정에 맞춰 빠른 동작 구현을 우선으로 진행한 전시용 시스템

---

## 문제 상황 & 해결 경험

**문제**: 박람회 전시 일정에 맞춰 미니어처를 실제로 동작시켜야 했으나, 하드웨어 제약 발생

**역할**: Python 기반 하드웨어 제어 로직 구현 / 외부 시스템 연동 API 구성

**결과**:
- 라즈베리파이 → 아두이노 기반 제어 방식으로 전환
- 제한된 일정 내 전시 시스템 완성
- 하드웨어 문제 구분 및 대응 경험

---

## 문제 상황 & 해결

### 하드웨어 제약을 고려한 제어 방식 전환

**현상**: 라즈베리파이 단독 제어에서 모터가 안정적으로 동작하지 않음

**원인**:
- 모터 구동에 필요한 전류 및 신호 제어 방식의 차이
- 소프트웨어가 아닌 하드웨어 궁합 문제

**해결**:

초기에는 라즈베리파이를 중심으로 미니어처 제어 로직을 구성했으나, 실제 구동 과정에서 모터 및 배선 등 하드웨어 특성으로 인해 안정적인 동작이 어려운 상황을 경험했습니다.

문제 원인이 소프트웨어가 아닌 하드웨어와의 궁합에 있음을 인지하고, 미니어처 제작사에서 제공한 아두이노 기반 제어 방식으로 전환했습니다.

**라즈베리파이–아두이노 연동**

제작사에서 제공한 아두이노 관련 자료를 바탕으로 미니어처 구동 로직을 재구성하고, 라즈베리파이와 아두이노를 함께 사용하는 구조로 수정했습니다.

제한된 일정 속에서 기존 계획을 고집하기보다 실제 전시 환경에서 안정적으로 동작하는 방식을 우선하여 반복 테스트와 빠른 수정으로 시스템을 완성했습니다.

**결과**:
- 박람회 전시 일정 내 시스템 완성
- 하드웨어 문제와 소프트웨어 문제를 구분하는 경험
