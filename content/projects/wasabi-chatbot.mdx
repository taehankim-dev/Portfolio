> **프로젝트 개요**
>
> 자회사 '사라팜'의 스마트팜 주력 작물인 와사비를 대상으로 한 AI 기반 챗봇 시스템입니다. 스마트팜 도입 초보 농가 및 고객에게 재배·관리 정보를 제공하기 위한 목적으로 개발했습니다.
>
> **프로젝트 성격**: 실험적 AI 챗봇 프로젝트로, 실제 고객 배포 전 내부 테스트 및 검증 단계에서 진행

---

## 문제 상황 & 해결 경험

**문제**: 와사비 재배 정보를 챗봇 형태로 제공하고, 이미지 분석 기능을 함께 제공 필요

**역할**: 풀스택 개발 / AI 모델(YOLOv8, Gemini) 연동 및 통합 설계

**결과**:
- 텍스트 질의응답 + 이미지 분석 기능 통합
- 프론트엔드–백엔드–AI 서버 간 연동 흐름 구성
- AI 기능을 서비스에 적용할 때의 구조적 제약 경험

---

## 구현 포인트

### 1. AI 기능을 서비스 흐름에 적용

텍스트 기반 질의 응답과 이미지 분석 기능을 하나의 챗봇 UX로 제공하기 위해, 프론트엔드–백엔드–AI 서버 간 요청 흐름을 구성했습니다.

AI 모델 자체보다, 실제 서비스에서 어떻게 연결되고 동작하는지를 이해하는 데 초점을 둔 실험적 경험이었습니다.

**프론트엔드–백엔드–AI 서버 연동**

- Vue 3: 사용자 인터페이스 및 채팅 UX
- Node.js: 텍스트 질의응답 API (Google Gemini AI 연동)
- Python: 이미지 분석 API (YOLOv8 + Google Gemini AI 연동)

### 2. 여러 기술 조합 탐색

Vue, Node.js, Python 기반 환경에서 서로 다른 기술을 연결하기 위해 다양한 예제와 문서를 참고하며 연동 방식을 시도했습니다.

이 과정에서 AI 기능을 서비스에 적용할 때 고려해야 할 구조적 제약과 난이도를 체감했습니다.
